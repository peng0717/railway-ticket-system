# WebTRS - 模拟铁路车站人工售票系统

基于 Python Flask 开发的模拟铁路客票发售和预订系统（TRS v5.0）的 Web 版本。

## 功能特性

### 核心功能
- **登录认证**: 工号+密码登录，选择白班/夜班
- **售票模块**: F1-F8键盘驱动，拼音码输入，席别选择，票价计算
- **退票模块**: 输入票号，自动计算手续费，确认退票
- **到达补票**: 无票/越站/越席/补差四种类型
- **余票查询**: 按车次查询、按发到站查询
- **交班结账**: 本班售票统计，实收金额汇总

### 界面特色
- 1:1 还原 TRS v5.0 操作界面
- 深色主题，专业的铁路售票系统风格
- 完整的键盘快捷键支持（F1-F8, F11, Alt+字母）
- 拼音码输入自动补全
- 实时余票信息显示

### 技术架构
- **后端**: Python Flask
- **数据库**: SQLite（零配置，文件型数据库）
- **前端**: Bootstrap 5 + 原生 JavaScript
- **认证**: Flask-Login

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化数据库

```bash
cd 铁路售票系统
python scripts/init_db.py
```

### 3. 运行应用

```bash
python app.py
```

### 4. 访问系统

打开浏览器访问: http://localhost:5000

## 车票预览与下载

售票成功后，点击"下载车票"按钮可以：
- 预览仿真车票图片
- 打印车票
- 下载车票图片（带"样票"水印）

## 注意事项

1. 本系统为教学/学习用途，生成的车票带有"样票"水印，不具备任何法律效力
2. 拼音码匹配采用前缀匹配方式，输入部分拼音即可搜索
3. 车票号采用7位格式（A+6位数字）
4. 退票手续费按开车前时间自动计算
